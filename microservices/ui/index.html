WS-ReportBot

Stay Updated with Latest Changes in your Github Repo!

Get Weekly Summary Report in a Slack #channel or via Direct Message.

Specially Meant for
Large Github Organisations

1 - Regular Weekly Summary Reports

Gives regular updates about latest changes made to your repository by analyzing merged PRs per week. Helps you Catch up fast when coming back after a long break & stay up-to-date! :D

2 - Enlists Top Contributors Every Week

Get list of top contributors weekly to appreciate the significant contributions they made to your project! Give them honor in the community channel, making them feel a sense of achievement :)

How to get Started?

Add to Slack, and then invite the @wsreportbot to the channels where you want the weekly reports to be generated for everyone.

/invite @wsreportbot #general

@wsreportbot will generate weekly reports about a github repository once you subscribe it to that repo.

About

WS-ReportBot was created for Hasura HackDev Challenge on HackerEarth by Mehul Kumar Nirala and Parul Aggarwal. It uses Hasura Database, filestore and Hasura Notify API to store the fetched data about a github repository daily and generate summary reports weekly.

Data comes from the Github's Repository Statistics API.
